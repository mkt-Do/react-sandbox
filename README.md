# React boiler-plate
This is React boiler plate (for individual(mkt\_mks))

## How to Use
First,
```
npm install
```
Next,
```
npm run build
or
npm start
```

## change code
if you change javascript code, write codes in '/src/js/' directory or css codes in '/src/css/' directory.  
You can use 'React', 'Redux' and 'FlowType'.
